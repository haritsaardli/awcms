# AWCMS Mobile

Multi-tenant Flutter mobile application for AWCMS.

## Tenant Folders

Each tenant has its own folder with complete Flutter project:

| Tenant | Path | Description |
| :--- | :--- | :--- |
| primary | [primary/](./primary/) | Default tenant |

## Quick Start

```bash
cd awcms-mobile/primary
flutter pub get
flutter run
```
